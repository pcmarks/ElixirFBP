ElixirFBP.Component

  • Summary
  • Functions

Component is used to start an FBP component. It takes the component’s name - an Elixir module name - and spawns a process. It is assumed that the component module supports a loop function which will be called when the component is spawned.

Component is also used to send data to a specific port on a specific node. The node has already been spawned and is identified my the atom representation of its node name.

send_ip(target, value)

Assemble and send an IP to an inport of a running process
start(graph_reg_name, node_id, component)

Spawn a process, identified by its component (module) name and with a process name of the atom value of graphid <> ““ <> node_id. This function can figure out a node’s inports and outports by accessing the respective function in the component module definition, e.g. Math.Add.inports Return the process name for the spawned process
stop(graph_reg_name, node_id)

Stop a component. This means find the process node’s pid, unregister it and force an exit
