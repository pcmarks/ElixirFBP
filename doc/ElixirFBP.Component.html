ElixirFBP.Component

  • Summary
  • Functions

Component is used to start an FBP component. It takes the component’s name - an Elixir module name - and spawns a process. It is assumed that the component module supports a loop function which will be called when the component is spawned.

start(component, node_id, inports, outports, no_of_processes \\ 1)
(function) # ↑

Start the execution of a component in its own process(es). Spawn as many processes as are specified in the no_of_processes value (the default value is one).

inports is a list of {port, value} tuples where value is an initial value or a Subscription pid. outports is a list of {port, pid} tuples where the pid is a Subscription.

inports and outports are used to create the initial arguments sent to a Component’s loop function. They are also used to create lists of component pids that must be sent to any Subscriptions that a component’s in or out ports are connected to.
