    ElixirFBP.ComponentLoader

    This module is responsible for loading ElixirFBP components at run time. Given a list of file paths, it will examine all modules, select those which implement the ElixirFBP.ComponentBehaviour and develop a list of Component attributes such as description, inports, and outports.

    This code is based on the answer to a question about loading Elixir modules. The code is from the hex package manager.
