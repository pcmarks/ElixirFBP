FBPNetwork.FBPHandler

FBPHandler contains functions that execute the Flow-Based Programming (FBP) Network Protocol commands. For more details see here

The protocol is message-based; the transport layer for the messages acts asynchronously. A message consists of three parts: a protocol, a command and a payload

There are four types of protocol:

  1. runtime
  2. network
  3. component
  4. graph

Within each protocol, there are several commands. See the approriate fbp_handle function for their descriptions.
